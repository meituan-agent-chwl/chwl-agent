# Meituan Local-Life Mock API

This is a zero-dependency Python Mock API for the Meituan hackathon demo.

It simulates a dynamic local-life fact sandbox:

- activity search
- restaurant search
- route estimation
- current weather
- live queue status
- booking availability
- external fulfillment entry
- user-confirmed fulfillment status
- background watch tasks
- sandbox events
- alternatives for partial replanning
- lightweight memory/profile defaults
- session memory written by frontend or agent events

The API does not make agent decisions. It only returns facts and changing state.
Intent parsing, feasibility checks, ranking, explanation, and partial replanning
belong to the Agent/Planner layer.

## Run

```powershell
python -B app.py
```

Open:

```text
http://127.0.0.1:8000/api/health
```

## Useful Demo Calls

Search family activities:

```text
http://127.0.0.1:8000/api/activities/search?scenario=family&radius_km=8
```

Search family restaurants:

```text
http://127.0.0.1:8000/api/restaurants/search?scenario=family&preferences=清淡,低油
```

Trigger restaurant queue spike:

```powershell
Invoke-RestMethod -Method Post -Uri http://127.0.0.1:8000/api/sandbox/trigger-event -ContentType 'application/json' -Body '{"event_type":"queue_spike","target_poi_id":"rest_family_001"}'
```

Poll events:

```text
http://127.0.0.1:8000/api/events/poll
```

Get replacement options:

```text
http://127.0.0.1:8000/api/alternatives/search?affected_node_id=rest_family_001&scenario=family&reason=queue_spike
```

Trigger heavy rain:

```powershell
Invoke-RestMethod -Method Post -Uri http://127.0.0.1:8000/api/sandbox/trigger-event -ContentType 'application/json' -Body '{"event_type":"weather_heavy_rain"}'
```

Reset sandbox:

```powershell
Invoke-RestMethod -Method Post -Uri http://127.0.0.1:8000/api/sandbox/reset
```

## LLM Event Simulator Hook

The API can accept an event generated by an LLM simulator. The LLM should only
simulate environment facts, not make planner decisions.

```powershell
Invoke-RestMethod -Method Post -Uri http://127.0.0.1:8000/api/sandbox/apply-llm-event -ContentType 'application/json' -Body '{"event_type":"queue_spike","target_poi_id":"rest_family_001","severity":"high","message":"晚餐高峰到来，餐厅等待时间升至65分钟","state_patch":{"queue":{"queue_tables":24,"estimated_wait_min":65,"can_take_number":true,"status":"queue_spike"}}}'
```

## Session Memory

The profile starts empty. The frontend or Agent writes facts after asking the
user or extracting explicit context from an event.

```powershell
Invoke-RestMethod -Method Post -Uri http://127.0.0.1:8000/api/memory/update -ContentType 'application/json' -Body '{"scope":"session_facts","source":"user_confirmed","updates":{"home_area":"北京望京","child_age":5,"spouse_diet":["减脂","清淡"]}}'
```
